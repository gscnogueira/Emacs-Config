# Emacs Config

Emacs 30.2. Configuração pessoal usando `use-package` com instalação automática via MELPA.

## Dependências externas

- **JetBrains Mono** e **Literata** — fontes
- **nerd-icons** — instalar com `M-x nerd-icons-install-fonts`
- **Node.js** — para o Copilot Language Server (`~/.npm-global/bin/copilot-language-server`)
- **pyright** e **ruff** — LSP e formatação Python
